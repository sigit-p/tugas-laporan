Panduan Penggunaan Panel Admin
1. Keamanan Login

Gunakan akun yang sudah didaftarkan. Gunakan tombol mata () untuk melihat password yang sedang diketik guna menghindari kesalahan input.

2. Menginput Nilai
  • Pilih menu INPUT NILAI sesuai mata pelajaran.
  • Pilih kelas, lalu cari nama siswa pada kolom pencarian (Autocomplete).
  • Pastikan Job dan Nilai sudah benar sebelum menekan tombol Simpan.
3. Mengedit & Memantau Nilai

Buka menu LIHAT & EDIT TABEL untuk melihat rekapitulasi nilai per kelas.

  • Klik tombol pada baris nama siswa untuk memperbaiki nilai yang salah.
  • Nilai di bawah 75 (KKM) akan otomatis ditandai dengan warna merah.
  • Gunakan tombol Refresh () jika data terbaru belum muncul di tabel.
Penting: Pastikan untuk selalu Logout setelah selesai bekerja untuk menjaga keamanan data nilai siswa.
